Efficient zero order optimization.

Simply open this file in browser to see the library work.

In order to see the source code, open the debugging view in your browser (e.g. F12 in Chrome), and inspect the sources for the examples.

A simple 2d quadratic function is used for minimization.

Black box optimization is concerned with optimization of functions which typically have no analyticall expression, are expensive to evaluate, and whose outputs are noisy.

In this example, such algorithms are compared.
